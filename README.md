# giga-groot

GIGA Groot web project for team reuse.

## Online Preview
GitHub Pages preview is enabled for this project.

## Project Structure
- `index.html` - main entry page
- `groot-agent.html` - secondary page
- `images/` - image assets
- `videos/` - video assets

## How to Use
1. Download ZIP or clone this repository
2. Open `index.html`
3. Edit images, videos, and HTML as needed

## Notes
- Keep asset paths unchanged when replacing files
- Put new images in `images/`
- Put new videos in `videos/`
